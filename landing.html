Features
What can Morpheus do?

From a single chat interface, Morpheus handles anything that used to require switching between apps.

📧
Read & draft emails
Search your inbox, read threads, compose replies — all from natural language. Sends only after you approve.
📅
Manage your calendar
Check availability, create events, reschedule meetings. Never double-books without confirmation.
📁
Search Google Drive
Find files by name or content across all your Drive folders. Create, edit, or share documents.

Remembers context
Conversation history stored locally. Follow-ups and commitments are automatically tracked.
⚡
Routes by intent
Simple questions get instant answers. Complex tasks go through the full agent pipeline — no wasted LLM calls.
🔒
Approval-first design
Every action that modifies external state — send, create, delete — surfaces a confirmation card before executing.

Approval Gate — built in, always on

Morpheus never sends emails, creates events, or deletes files without surfacing a draft first. Review exactly what will happen, then approve or discard.

  • Email sends, replies, and deletions
  • Calendar event creation, modification, and deletion
  • Google Drive file deletion
  • Any action marked as irreversible

Architecture
How it works

Morpheus mirrors how a real executive assistant operates — classify first, route intelligently, always confirm before acting.

1

Intent Classifier

Every message is classified in <300ms using a fast model. Returns intent, priority, confidence, and language.

email · calendar · task · research · simple_response
2

Intent Router

High-confidence simple intents go directly to Quick Response — no tool calls, lower cost. Everything else routes to the Main Agent.

3

Main Agent + Tool Calling

The agent uses Gmail, Calendar, and Drive tools to gather context and prepare actions. Read-only tools execute immediately. Write operations are staged for approval.

4

Parallel: Follow-up Detector

After every response, a background process scans the exchange for commitments, deadlines, or pending replies — and logs them automatically.

5

Local Storage

All memory — messages, tasks, follow-ups — is stored in JSON files inside ~/.morpheus/. No cloud, no sync, no account required.

Or run in Terminal:

# Build frontend + start
$ npm run build && npm start

# Development mode (hot reload)
$ npm run dev
